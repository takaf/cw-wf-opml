# cw-wf-opml
Convert selected Chatwork messages to OPML for Workflowy.

## What it does
- Select text in Chatwork
- Right-click → **Workflowy用にコピー**
- Paste into Workflowy (OPML)

## Install (unpacked extension)
1. Download or clone this repository.
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode**
4. Click **Load unpacked** and select this folder.

## Usage
1. Open Chatwork in your browser.
2. Select a range of text in a message.
3. Right-click and choose **Workflowy用にコピー**
4. Paste into Workflowy.

## Notes
- This project is not affiliated with Chatwork or WorkFlowy.
- If Chatwork changes its DOM, message-id detection may stop working.
